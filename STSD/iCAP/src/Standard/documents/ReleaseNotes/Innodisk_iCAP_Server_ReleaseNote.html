Server Installation

Pre-install:

  • Before you install iCAP Server images, makesure you already installed Docker server on your server.
  • If you need more information for installation of Docker server, reference to the Docker offical installaition guide: https://docs.docker.com/get-docker/

Installation step (Linux):

  1. Decompress the tar file: tar zxvf iCAP_Server_linux_V_1_6_1.tar.gz
  2. Switch into the decompress iCAP_Server_linux_V_1_6_1 folder
  3. Run the installation: sudo ./iCAP_Server_Installer
  • After installation:
  • You can use this command to check iCAP Server running status : sudo docker ps -a

Installation step (Windows):

  1. Decompress the zip file: iCAP_Server_windows_V_1_6_1.zip
  2. Switch into the decompress iCAP_Server_windows_V_1_6_1 folder
  3. Run the installation: iCAP_Server_Installer.exe
  • After installation:
  • You can use this command to check iCAP Server running status : docker ps -a

For more information of installation, please reference the iCAP Installation guide.
