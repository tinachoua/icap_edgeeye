1. Description

After makeing on the linux, execute the build_iCAP_Server_Installer.bat to build the icap server installer fo windows.

2. Parameter

  -cr
  	Further Information: 
	  Compress archive for Release Version.
	  ex : 
	  .\iCAP_Server_Packer_Windows.bat cr

  -cd
  	Further Information: 
	  compress archive for DemoKit Version .
	  ex : 
	  .\iCAP_Server_Packer_Windows.bat cd

  -b
  	Further Information: 
	  Build the iCAP Server Installer for Windows 
	  ex : 
	  .\iCAP_Server_Packer_Windows.bat b
